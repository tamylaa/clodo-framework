## Sponsor tiers (example)
These are suggested tiers. You can adapt naming and benefits.

- Bronze — $5/month
  - Public thanks in `README.md` and `CONTRIBUTORS.md`
- Silver — $25/month
  - All Bronze benefits + quarterly sponsor call and README badge
- Gold — $100/month
  - All Silver benefits + roadmap prioritization, company logo on README, quarterly demo

If you prefer to sponsor a specific feature or pay for a bugfix, please open an issue titled `SPONSOR: <short description>` and include your contact details.

## How contributions are used
We commit to financial transparency. Funding will be used for:
- Continuous Integration (CI) and test environments
- Documentation and example services
- Bug fixes and small feature work
- Sponsorship recognition and community events

We will publish periodic summaries of funded spend in this repository (monthly updates in `FUNDING.md`).

## How to set up payments (quick guide)
- GitHub Sponsors: Create a GitHub Sponsors profile and add your repository to it.
- Open Collective: Create a collective and either use a fiscal host or set up your own organization.
- Buy Me a Coffee / Patreon: Sign up and copy your page link into this document.
- PayPal.me: Create a PayPal.Me link or business donate button and copy the link here.
